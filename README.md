# Todo list using basic HTML, JavaScript and CSS

Demo here: https://fir-todo-95078.firebaseapp.com/

A HTML, JS and CSS todo list using:
- Local storage
- Webpack
- MomentJS
- Babel as compiler

## How to use it
- Clone it
- $ npm install
- $ npm run webpack (for dev, or)
- $ npm run webpack:prod (for deployment)
